SEA_get_provider()

iSEA_API.php : 49

GET or INIT provider dataset

public SEA_get_provider([int &$status = 0 ]) : array<string|int, mixed>

If the current one is SEA User (role) and it's the first call, therefore there are no provider associated for this user, it creates a new record in table WP_SEA_providers AND returns it (with the ID) empty.

Otherwise, if there is a provider associated with this user, it returns all associated data in only one recorset: User-Provider-Locations and so on.

Parameters
$status : int = 0
Return values
array<string|int, mixed> —

provider | false if ERROR
