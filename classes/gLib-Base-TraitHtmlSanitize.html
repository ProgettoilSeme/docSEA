TraitHtmlSanitize.php : 16

Trait trait TraitHtmlSanitize

Questo trait fornisce metodi per sanificare stringhe HTML, rimuovendo tag e classi CSS non consentiti in base a regole predefinite. È progettato per garantire la compatibilità con l'editor Quill e per prevenire l'inserimento di contenuti non validi o potenzialmente dannosi. Include anche metodi per verificare il tipo di contenuto (HTML, JSON, serializzato, ecc.).

Sanifica un testo HTML compatibile con Quill.

public sanitizeHtml([string $input = '' ][, string|null &$err = '' ]) : string|false

Questo metodo prende in input una stringa HTML e la sanifica rimuovendo eventuali tag o classi CSS non consentiti, secondo le regole definite nelle proprietà $allowed_tags e $allowed_classes. Se il contenuto non è valido o contiene elementi non ammessi, restituisce un errore descrittivo tramite il parametro $err.
